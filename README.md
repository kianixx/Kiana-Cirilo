# My Portfolio

A personal portfolio website built with **React**, **Vite**, and **Tailwind CSS**, showcasing my education, skills, achievements, projects, and contact information.  

This project demonstrates a clean, responsive, and interactive front-end design.

---

## 🛠 Technologies Used

- [React](https://reactjs.org/) – Front-end library  
- [Vite](https://vitejs.dev/) – Build tool for fast development  
- [Tailwind CSS](https://tailwindcss.com/) – Utility-first CSS framework  
- [SVG / PNG Icons](https://lucide.dev/) – Social media icons
